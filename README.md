# gRPC-Practice

Pre-requisite:
- `npm i` to install all dependencies

For server: <br>
`cd server` <br>
`npm start`

For client: <br>
`cd client` <br>
`node index.js`

Open web browser and go to `localhost:3000` to see the result. <br>
